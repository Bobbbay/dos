# Dos
DDoS written in GoLang.

# Install
```
git clone https://github.com/Bobbbay/dos.git
```
```
cd dos
```
```
make install
# --- or ---
bash install.sh
```

# Update
```
make update
# --- or ---
git pull origin master
bash install.sh
```
